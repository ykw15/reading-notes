⚖️ 炎症的两面性

✅ 急性炎症（有益）

  • 红、肿、热、痛明显
  • 快速出现，快速消退
  • 保护性反应
  • 免疫系统正常运作

❌ 慢性炎症（有害）

  • 无明显症状（"闷烧"）
  • 长期持续，逐渐累积
  • 破坏性反应
  • 免疫系统失调

更多疾病与炎症的关联

抑郁症、癌症、痴呆的炎症机制

🧠 抑郁症

慢性炎症 → 炎症因子进入大脑 → 影响神经递质合成（血清素减少） → 情绪调节中枢功能异常 → 抑郁症状

核心观点：抑郁症也是一种炎症性疾病

🎗️ 癌症

长期慢性炎症 → 细胞持续损伤 → DNA损伤累积 → 细胞变异 + 炎症环境提供生长温床 → 癌细胞增殖、转移

🧓 痴呆（阿尔茨海默病）

脑部慢性炎症 → 神经细胞损伤 → tau蛋白异常磷酸化 → 神经纤维缠结 → 认知功能下降

🤧 过敏性疾病

过敏原进入体内 → 免疫系统过度反应 → 产生抗体 + 释放炎症介质 → 炎症反应 → 过敏症状（花粉症、哮喘、皮炎）

本质：过敏是慢性炎症的表现形式

常见问题 Q&A

解答你的疑惑

Q1: 花生过敏和炎症有什么关系？

两者都是免疫系统的过度反应。过敏是免疫系统对无害物质产生过激防御，本质上与慢性炎症相关。

Q2: 牙龈出血和心脏病有什么关系？

牙周病的炎症因子会进入血液循环，影响全身血管健康，研究显示牙周炎与心血管疾病密切相关。

Q3: 压力大真的会导致炎症吗？

是的。慢性压力会导致皮质醇长期升高，抑制免疫功能，同时增加促炎因子分泌。

Q4: 减肥需要达到什么目标？

BMI控制在18.5-24范围内，特别是减少内脏脂肪（腹部肥胖）对抗炎最重要。

Q5: 鱼油和亚麻籽油有什么区别？

鱼油含EPA和DHA（直接可用），亚麻籽油含α-亚麻酸（需在体内转化，转化率约5-10%）。两者都推荐。
